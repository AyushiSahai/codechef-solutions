# 🍴 CodeChef Solutions

A clean, organized collection of my CodeChef problem solutions in Python.

## 📁 Folder Structure

Easy/      → Easy difficulty problems
Medium/    → Medium difficulty problems
Hard/      → Hard difficulty problems
Contests/  → Contest solutions
templates/ → Copy solution_template.py for every new problem

## 🚀 Daily Workflow

1. Copy template:
   cp templates/solution_template.py Easy/PROBLEM_CODE.py

2. Solve the problem

3. Upload to GitHub:
   git add .
   git commit -m "Add: PROBLEM_CODE - Problem Name (Easy)"
   git push
